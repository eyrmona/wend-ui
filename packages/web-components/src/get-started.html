wend-ui
Get Started Foundations Components

Installation

Install the web components package, plus the base styles package for tokens/reset/utilities:

npm install @wend-ui/web-components @wend-ui/styles

Basic usage

Load the component loader script and the base stylesheet, then use any wend-ui element directly in HTML:

<link rel="stylesheet" href="node_modules/@wend-ui/styles/dist/index.css" />
<script type="module" src="node_modules/@wend-ui/web-components/loader/index.js"></script>

<wend-button variant="primary">Click me</wend-button>

Using with React

A generated set of React wrapper components ships from @wend-ui/react, so components can be used the same way as any other React component, with full prop typing:

import { WendButton } from '@wend-ui/react';

function App() {
  return <WendButton variant="primary">Click me</WendButton>;
}
